# ⚠ Do not file security vulnerabilities as public issues.

Public GitHub issues are visible to everyone, including attackers.
Filing a security vulnerability here could expose users to harm while
we work on a fix.

**Please close this issue without filing.** Instead, disclose the
vulnerability through one of the channels described in `SECURITY.md`:

1. **Email** — `open.source@pointsav.com`
2. **GitHub Security Advisories** — click the **Security** tab of this
   repository, then **Report a vulnerability**. This creates a private
   advisory visible only to maintainers.

Our coordinated-disclosure commitments are in `SECURITY.md`.

Thank you for helping keep users of Woodfine-managed software safe.
